# MISW4203-202415-ing-de-software-para-aplicaciones-moviles

# compilar proyecto
Descargar proyecto(git clone https://github.com/orlandoarnedouniandes/MISW4203-202415-ing-de-software-para-aplicaciones-moviles.git)

Abrir el proyecto en Android Studio

Build - Make Project

En caso de no tener un Device manager, crear uno o conectar un celular vis USB. 

Dar clic en Run app para que se ejeute la aplicación sobre el dispositivo

# Ejecutar Pruebas automatizadas
Las pruebas se encuentran dentro del proyecto en /app/kotlin+java/com.mobileapp.mymobileapp.mymobileapp(Android Test)
clic derecho y "run Tests in com.mobile..."
